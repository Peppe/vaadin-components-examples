v-grid Columns

Defining columns

Columns can be defined using HTML or Javascript.

For the HTML version, the table element might seem redundant, but it is required in order for the browsers to parse the contained markup correctly. We also need to provide some data for the grid, using tbody, in order to render something.

Read the column API documentation to see all the attributes and properties you can define for a column.

One Two
grid.columns = [ { headerContent: "First column" }, { headerContent: "Second column" } ];

Adding columns

Columns can be added dynamically using the Javascript API. You can also modify the lightDom of the component, it is not the recommended way.

Since some of the API is asynchronous, v-grid provides the then() function which can be used to trigger a callback after any pending tasks are finished (the addition of the column 'Column one' in this case).
