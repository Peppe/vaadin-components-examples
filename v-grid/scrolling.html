v-grid Scrolling

Scrolling the Grid

JS API provides functions for scrolling the Grid programmatically.

  • scrollToStart() restores the scroll position back to the beginning.
  • scrollToEnd() scrolls the Grid all the way to the last data row.
  • scrollToRow(index) allows scrolling to a specific row index.
